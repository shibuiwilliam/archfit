# Remediation: P1.LOC.003 — High dependency coupling

**Summary**: A package has transitive reach exceeding 10, meaning agents must load many neighbors to understand changes there.

## Decision tree

1. **Is the high-reach package a shared types/model package?**
   - Yes (e.g., `internal/model/`) → these are read-heavy, rarely changed. Suppress with rationale.
   - No → continue.

2. **Can the package be split?**
   - Extract interfaces at module boundaries.
   - Move shared types into a dedicated, narrower package.
   - **Ask the user** for guidance on which responsibilities to separate.

3. **Re-scan** to confirm reduced coupling.
