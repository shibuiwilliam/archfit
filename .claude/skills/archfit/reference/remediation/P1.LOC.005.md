# Remediation: P1.LOC.005 — High-risk path lacks INTENT.md

**Summary**: A directory matching a high-risk keyword (`auth`, `secret`, `migration`, `deploy`) has no `INTENT.md`.

## Decision tree

1. **Is the directory actually high-risk?**
   - No (false positive from naming) -> suppress with `.archfit.yaml` ignore entry.
   - Yes -> continue.

2. **Create `INTENT.md`** in the flagged directory with:
   - One sentence on what the directory does.
   - Invariants that must hold after any change.
   - Links to relevant specs or ADRs.

3. **Re-scan** to confirm the finding is gone.

## Minimal INTENT.md template

```markdown
# <directory-name> — Intent

## Purpose

<one sentence>

## Invariants

- <invariant 1>
- <invariant 2>

## See also

- <link to spec or ADR>
```
