# Remediation: P1.LOC.009 — Missing runbook

**Summary**: The repository has deployment artifacts but no `RUNBOOK.md` at the root or in `docs/`.

## Decision tree

1. **Does a runbook exist elsewhere (wiki, external doc)?**
   - Yes -> either create a stub `RUNBOOK.md` linking to it, or suppress with ignore entry.
   - No -> continue.

2. **Create `RUNBOOK.md`** at the repository root with:
   - Deployment procedure.
   - Rollback procedure.
   - On-call escalation path.

3. **Re-scan** to confirm the finding is gone.

## Minimal RUNBOOK.md template

```markdown
# Runbook

## Deploy

1. <step 1>
2. <step 2>

## Rollback

1. <step 1>

## Escalation

- On-call: <team or person>
```
