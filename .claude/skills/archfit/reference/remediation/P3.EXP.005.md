# Remediation: P3.EXP.005 — Global mutable state minimized

**Summary**: The repository has too many mutable package-level variables. Move state into struct fields or function parameters for explicit data flow.

## Decision tree

1. **Check the finding evidence** for the list of flagged variables.
2. **Can the variable be a constant?**
   - Yes -> convert `var` to `const`.
3. **Is the variable used across functions in the package?**
   - Yes -> move it into a struct and pass the struct explicitly.
   - No -> make it local to the function that uses it.
4. **Re-scan** to confirm the count drops below the threshold.

## Example fix

```go
// Before: package-level mutable var
var defaultTimeout = 30 * time.Second

func doWork() {
    client := &http.Client{Timeout: defaultTimeout}
}

// After: struct field
type Config struct {
    Timeout time.Duration
}

func doWork(cfg Config) {
    client := &http.Client{Timeout: cfg.Timeout}
}
```

## When to suppress
