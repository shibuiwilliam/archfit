# Remediation: P4.VER.003 — No CI configuration

**Summary**: The repo has source code but no CI configuration. It is locally verifiable but not continuously verified.

## Decision tree

1. **Is this a personal/experimental repo?**
   - Yes → suppress with rationale.
   - No → continue.

2. **Which CI platform does the team use?**
   - **Ask the user** — do not guess between GitHub Actions and GitLab CI.

3. **Create a minimal CI config** that runs the existing test command.

4. **Re-scan** to confirm.

## GitHub Actions example

```yaml
name: ci
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: make test
```
