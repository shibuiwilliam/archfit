# Remediation: P1.LOC.002 — Slice lacks AGENTS.md

**Summary**: A directory under a known slice container has no `AGENTS.md` or `CLAUDE.md`. Recognized containers: `packs/`, `services/`, `modules/`, `packages/`, `apps/`, `libs/`, `plugins/`, `engines/`, `components/`, `domains/`, `features/`.

## Decision tree

1. **Is the slice legitimate, or is it a leftover?**
   - Leftover / deprecated → propose deleting it; don't paper over it with an AGENTS.md. **Ask the user** before deleting.
   - Legitimate → continue.

2. **Is the slice trivially small (one short file)?**
   - Yes → propose `.archfit.yaml` suppression with `reason: "single-file slice"`, `expires: <6 months>`.
   - No → continue.

3. **Create `AGENTS.md` at the slice root** using the template below. Keep it terse — link to repo-level docs, do not duplicate.

4. **Re-scan** to confirm the finding is gone for that slice.

## Minimal slice AGENTS.md template

```markdown
# <slice-name> — agent guide

## What this slice is

<one sentence>

## Invariants (do not break)

- <invariant 1>
- <invariant 2>

## Fast verification (this slice only)

```bash
<command>
```

## Do not

- <thing that is tempting but wrong>

## Links

- Repo-level contract: ../../CLAUDE.md
- Owner: <person or team>
```
