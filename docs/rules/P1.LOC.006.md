# P1.LOC.006 — Agent-facing documentation is not bloated

| Field | Value |
|---|---|
| Principle | P1 (Locality) |
| Severity | `warn` |
| Evidence | `strong` |
| Stability | `experimental` |

## What it checks

CLAUDE.md and AGENTS.md must not exceed 400 lines or 10 KB. Files beyond
these limits may not fit in an agent's context window, defeating the purpose
of having a canonical entry point.

## How to fix

Split the document. Move reference material, deep architecture notes, and
runbooks into linked files. Keep the root document as a concise entry point.

## Suppressing this rule

```yaml
ignore:
  - rule: P1.LOC.006
    reason: "Monorepo requires comprehensive root docs"
    expires: 2027-01-01
```
